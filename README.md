# 🎓 Student Management System

A Spring Boot based **Student Management System** with **JWT authentication** and **role-based authorization**.  
The system supports **ADMIN**, **STUDENT**, and **TEACHER** roles with secure login and access control.

---

## 🚀 Features

- User Registration & Login
- JWT-based Authentication
- Role-based Authorization (ADMIN / STUDENT / TEACHER)
- Secure Password Encoding (BCrypt)
- Spring Security Integration
- JPA & Hibernate ORM
- MySQL Database
- RESTful APIs

---

## 🛠️ Tech Stack
**Java**
- **Spring Boot**
- **Spring Security**
- **JWT (JSON Web Token)**
- **Spring Data JPA**
- **Hibernate**
- **MySQL**
- **Maven**
- **Lombok**
